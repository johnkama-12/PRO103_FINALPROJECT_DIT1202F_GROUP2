# PRO103 Final Project - DIT1202F Group 2

## Student Academic Portal Management System

This repository contains the final project submission for **PRO103 / Principle of Structured Programming** by **DIT1202F Group 2**. The project is a **Student Academic Portal Management System** developed using Python. It provides a simple academic platform where students can register, log in, manage their subjects, view assignments, grades, calendar events, and other academic information.

## Project Overview

The Student Academic Portal Management System was designed to help students organize their academic activities in one place. The system allows users to create an account, log in securely, and access academic features based on the subjects they entered during registration.

The system demonstrates the use of structured programming concepts such as variables, data types, functions, conditional statements, loops, lists, dictionaries, file handling, and modular program design.

## Main Features

* Student signup and login system
* Students can enter their own subjects during registration
* Demo accounts with default courses and data
* Student dashboard with academic information
* Course and subject display
* Assignment management
* Grade display
* Calendar section
* Message section
* Student data saved using JSON file storage
* Graphical user interface using CustomTkinter
* Project documentation included in PDF and Word format
* Screenshots included as evidence of system design and execution

## Technologies Used

* Python
* CustomTkinter
* JSON

## Requirements

Before running the project, make sure Python is installed on your computer.

Recommended version:

```text
Python 3.10 or above
```

Install the required Python packages using:

```bash
pip install -r requirements.txt
```

## How to Run the Project

Open the project folder in your terminal or command prompt and run:

```bash
python source_code/now.py
```

The application window will open and allow users to log in or register.
